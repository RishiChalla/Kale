Detailed Description

template<typename S>
class Kale::FiniteStateAnimatable< S >

All nodes which allow for state based animation must inherit from this class. This is an implementation of a Finate State Machine. There are saved hardcoded "states" in the animation where the data for each state is saved in structure S. It is the responsibility of the inherited node to lerp between any states as required.

Template Parameters
Astructure representing the data required for a single state of the animation
