Detailed Description

A node capable of skeletal based animations. A single skeletal animatable node can only be used by a single node. This can be used to make unique "warp" effects which are otherwise impossible.

Please see the SkeletonNode if you are looking for a skeletal system capable of connecting several node's transforms.
